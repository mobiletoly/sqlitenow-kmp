# SQLiteNow Documentation

This directory contains the Jekyll-based documentation website for SQLiteNow.

## Local Development

To run the documentation site locally:

```bash
cd docs
./run-local.sh
```

The site will be available at `http://localhost:4000/sqlitenow-kmp/`

The root URL redirects to the Kotlin Multiplatform path:
`http://localhost:4000/sqlitenow-kmp/kmp/`

## Structure

- `index.html` - Root redirect to the Kotlin Multiplatform docs
- `flutter/` - Flutter and pure Dart setup/runtime/reference/recipe/sync docs
- `kmp/` - Kotlin Multiplatform setup/reference/recipe/migration/sync/tutorial docs
- `swift/` - Native Swift generated-package setup/reference/recipe/migration/sync docs
- `_includes/shared/` - Shared prose rendered by platform-specific pages
- `sync/` - Shared oversqlite concepts and server docs
- `_config.yml` - Jekyll configuration

## Deployment

The site is configured to be deployed to GitHub Pages at:
`https://mobiletoly.github.io/sqlitenow-kmp/`

## Navigation

The site uses the Minima theme with a custom header. The header renders a
platform selector row and a platform-scoped main menu:

- Flutter/Dart: Home, Getting Started, SQL Reference, Recipes, Migrations, Sync
- KMP: Home, Getting Started, SQL Reference, Recipes, Migrations, Sync, Tutorials
- Swift: Home, Getting Started, SQL Reference, Recipes, Migrations, Configuration, Sync

Shared topic content lives in `_includes/shared/` and is rendered by thin pages
under `flutter/`, `kmp/`, and `swift/`.

Do not hand-edit `_site`; it is generated by Jekyll.
